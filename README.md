# TippyCam

TippyCam is a minimal iPhone camera for bright, vivid everyday photos. It keeps capture simple, uses the native iPhone camera pipeline, applies one on-device render, and saves the result to Photos without accounts, subscriptions, or cloud processing.

## Highlights

- Native iPhone camera capture
- Bright, vivid on-device photo rendering
- Photo and Portrait modes
- Front and rear camera support
- In-app recent photo gallery
- Private, on-device capture, rendering, and saving
